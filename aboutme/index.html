Projects

  • Arcade Machine

    A cocktail cabinet arcade machine made using the lubuntu operating system on an emachines pc. Roms are run through the MAME emulator. Consists of games such as space invaders, ms. pacman, donkey kong, super mario bros, and punchout

  • Hypervisor 1 Server

    In an effort to save space on my macbook pro, I created a hypervisor 1 server using a mini pc in my basement. It runs a proxmox os containing the virtual machines of windows 10 and ubuntu 20.04. Windows 10 also has GPU passthrough and is accessed through Parsec to ensure frame rates of 60fps.
